//Crear Repositorios

Crear el repositorio directamente desde GitHub CLI 

1. Descargar 
    https://cli.github.com/

2. Autenticar 

    gh auth login

  * Selecciona en el menu:

    ? Where do you use GitHub? GitHub.com
    ? What is your preferred protocol for Git operations on this host? HTTPS
    ? Authenticate Git with your GitHub credentials? Yes
    ? How would you like to authenticate GitHub CLI? Login with a web browser
    
    ! First copy your one-time code: XXXX-XXXX
    Press Enter to open https://github.com/login/device in your browser... 
    ✓ Authentication complete.
    - gh config set -h github.com git_protocol https
    ✓ Configured git protocol
    ✓ Logged in as FredyxD25

3. Crear carpeta

    mkdir nombre-del-proyecto
    cd nombre-del-proyecto
    echo "# Primer Commit" > README.md
    git init
    git add .
    git commit -m "Primer commit"

4. Una vez logeado y con la carpeta creada
    
    gh repo create nombre-del-repo --public --source=. --remote=origin --push
    gh repo create Tauri --public --source=. --remote=origin --push
